Install Symfony
Below Command install the Symfony in our system using Composer.
Command: composer create-project symfony/website-skeleton my_project_name
  1. After install the Symfony appilcation. we need to add the symfony application in Git hub by using below commands
    Command: git init (it is used to initiate the git repository)
  2. Check the status of Git hub.
    Command: git status
  3. add the project by using below command in git hub.
    Command: git add .( . stand for every thing need to commit)
  4. To commit the application use the below command in git hub.
    Command: git commit -m "init" (m stand for message)
  5. Run the Symfony Application use below command.
    Command: php -S 127.0.0.1:8000 -t public
  6. List of console command.
    Command: php bin/console
  7. Check Router path for symfony application.
    Command: php bin/console debug:router
  8. Add the server namespace default command.
    Command: composer require server --dev

Route the Symfony path to Symfony page

  1. Create New ".twig" file inside the template folder(project/templates/filename.html.twig)
  2. Add the below code in routes.yaml (project/config/routes.yaml)
    filename is hello_page hello_page: path: / controller: Symfony\Bundle\FrameworkBundle\Controller\TemplateController::templateAction defaults: template: 'hello_page.html.twig'
  3. Run the the command to check the Router is set or not line.
    php bin/console debug:router
